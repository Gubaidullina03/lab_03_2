## 8 вариант
## Задание 3.2.8. Напишите функцию для подсчета количества уникальных слов в тексте (без учета регистра).
## Решение:
Для решения данной задачи, нам небходимо привести текст к нижнему регистру и разбить его на слова с помощью .lower() и .split(). Затем очистить слова от знаков препинания: cleaned_word = word.strip('.,!?;:"()[]'). Подсчет уникальных слов производим в виде: return len(set(cleaned_words))

Получим:

```
def count_unique_words(text):

    words = text.lower().split()
    
    cleaned_words = []
    for word in words:
        cleaned_word = word.strip('.,!?;:"()[]')
        if cleaned_word: 
            cleaned_words.append(cleaned_word)

    return len(set(cleaned_words))

text = "Это пример текста для примера подсчета уникальных слов. Текста для примера."
print(count_unique_words(text))
```
